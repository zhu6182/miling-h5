# 「命里」小程序发展规划

> **核心策略**：前期免费积累用户 → 达到一定用户量后开启变现
> **用户目标**：首先达到 1000+ 活跃用户，再考虑付费功能

---

## 一、现状分析

### 1.1 当前功能模块
| 功能 | 状态 | 使用频率 |
|------|------|----------|
| 紫微斗数排盘 | ✅ 已实现 | 低频（一次性） |
| 八字排盘 | ✅ 已实现 | 低频（一次性） |
| AI解读 | ✅ 已实现（默认免费） | 低频（一次性） |
| 双人匹配 | ✅ 已实现 | 低频（偶尔） |
| 帮查功能 | ✅ 已实现 | 低频（偶尔） |
| 人生K线 | ✅ 有入口 | 中频 |

### 1.2 数据现状
- **用户模型**：仅有基本字段（openid、nickname、avatar、phone）
- **VIP系统**：未实现
- **支付系统**：未实现
- **广告系统**：有按钮但未真正接入
- **后台管理**：完全缺失
- **用户统计**：缺失

### 1.3 核心问题
1. **使用频率低**：命盘排好后，用户没有回来看的理由
2. **缺乏运营数据**：无法了解用户行为、留存率、活跃度
3. **变现时机未到**：用户量不足时开启付费会阻碍增长

## 六、每日运势详细设计

### 6.1 运势算法（简化版）

```python
# app/services/fortune_service.py

def calculate_daily_fortune(chart: Chart, date: str) -> dict:
    """
    基于命盘和日期计算每日运势
    
    输入：
    - chart: 用户命盘数据
    - date: 当日日期 "2024-01-15"
    
    算法步骤：
    1. 获取当日干支（年月日时四柱）
    2. 计算当日五行分布
    3. 对比命盘五行（喜忌分析）
    4. 计算命宫星曜当日状态
    5. 综合评分生成运势
    """
    
    # 1. 当日干支
    day_ganzhi = get_day_ganzhi(date)  # 如 "甲子"
    day_wuxing = ganzhi_to_wuxing(day_ganzhi)  # 如 "木水"
    
    # 2. 命盘五行分析
    chart_wuxing = chart.five_elements  # 如 "土五局"
    
    # 3. 计算五行相生相克关系
    score = calculate_wuxing_score(day_wuxing, chart_wuxing)
    
    # 4. 流日运势因子
    day_factor = get_day_lucky_factor(date, chart.soul_palace)
    
    # 5. 综合评分
    overall_score = int(base_score + score + day_factor)
    
    # 6. 分维度评分
    love_score = calculate_love_fortune(chart, date)
    career_score = calculate_career_fortune(chart, date)
    wealth_score = calculate_wealth_fortune(chart, date)
    
    # 7. 生成宜忌列表
    do_list = generate_do_list(overall_score, day_wuxing)
    avoid_list = generate_avoid_list(overall_score, day_wuxing)
    
    # 8. 幸运元素
    lucky_color = get_lucky_color(day_wuxing)
    lucky_number = get_lucky_number(date)
    
    # 9. 运势短语
    phrase = generate_fortune_phrase(overall_score)
    
    return {
        "overall_score": overall_score,
        "love_score": love_score,
        "career_score": career_score,
        "wealth_score": wealth_score,
        "lucky_color": lucky_color,
        "lucky_number": lucky_number,
        "do_list": do_list,
        "avoid_list": avoid_list,
        "phrase": phrase
    }
```
