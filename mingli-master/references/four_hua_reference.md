# 四化飞星参考

四化是紫微斗数的核心动力系统。每个天干对应四颗星的"禄权科忌"变化。

## 四化含义

| 四化 | 核心含义 | 类比 | 影响方向 |
|------|---------|------|---------|
| **化禄** | 财禄、顺利、享受 | 天上掉馅饼的好运 | 正面，增加资源 |
| **化权** | 权力、掌控、主导 | 拿到话筒就能主导全场 | 中性偏正，增加控制力 |
| **化科** | 名声、学术、贵人 | 被点名表扬的那个优秀学生 | 正面，增加名声 |
| **化忌** | 执着、阻碍、消耗 | 永远在想"为什么是我"的那个心结 | 负面，但也代表在意 |

## 四化落在不同宫位的解读

### 化禄

| 宫位 | 解读方向 |
|------|---------|
| 命宫 | 天生有福气，做事相对顺利 |
| 财帛宫 | 财运好，赚钱能力强 |
| 官禄宫 | 事业上有贵人，容易得到机会 |
| 夫妻宫 | 感情甜蜜，伴侣对你好 |
| 迁移宫 | 出门遇贵人，适合外出发展 |

### 化权

| 宫位 | 解读方向 |
|------|---------|
| 命宫 | 天生有主导权，喜欢掌控局面 |
| 官禄宫 | 事业上有话语权，适合做领导 |
| 财帛宫 | 理财能力强，能守住财富 |
| 夫妻宫 | 在感情中占主导地位 |

### 化科

| 宫位 | 解读方向 |
|------|---------|
| 命宫 | 有学术气质，容易出名 |
| 官禄宫 | 事业上有名声，适合学术类 |
| 父母宫 | 与父母关系好，受家庭教育影响大 |
| 福德宫 | 精神世界丰富，有文化修养 |

### 化忌（最重要！）

化忌是紫微斗数中最重要的分析点。它代表**执着、在意、消耗**——你在哪里化忌，就在哪里纠结最多。

| 宫位 | 解读方向 |
|------|---------|
| 命宫 | 对自己不满意，自我要求高但也容易自我否定 |
| 官禄宫 | 事业上波折多，容易遇到阻碍 |
| 财帛宫 | 钱财上容易有损耗，理财需谨慎 |
| 夫妻宫 | 感情中容易纠结，对伴侣要求高 |
| 福德宫 | 精神层面焦虑多，容易过度思考（中州派特别重视此位置） |
| 疾厄宫 | 健康上需要注意，定期体检 |
| 迁移宫 | 出门容易遇到不顺，外出需谨慎 |

## 中州派化忌追踪

中州派特别重视化忌的串联效应：

1. **生年化忌**：先天的执着点，一辈子都在这个领域纠结
2. **大限化忌**：这个十年的主要困扰和课题
3. **流年化忌**：当年的具体挑战

当生年化忌 + 大限化忌 + 流年化忌落在同一宫位时，该宫位的问题会被放大，需要特别注意。

## 解读技巧

1. **先找化忌**：化忌是命盘中最容易出问题的点，也是最有故事性的部分
2. **再看化禄**：化禄是你的资源所在，可以用来化解化忌的消耗
3. **化权化科为辅**：化权增加你的掌控力，化科增加你的名声和贵人缘
4. **四化叠加**：当生年四化和大限/流年四化叠加时，影响会被放大
